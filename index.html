# Sett

Reliable video streaming over unreliable networks.
Open source. Commercially supported.

// what we do ------------------

We build tools for teams who need video to work when conditions don't cooperate - low bandwidth, unstable connections, remote locations.

We've spent years working with the constraints of real-world deployments. We know what breaks, and we've built software to stop it breaking.

// you might know our work ------------------

  • SlingShot - ultra-low latency video streaming over QUIC
  • Dart - v4l2 (hdmi to csi, webcam, uvc) to rtsp server, replace EOL hardware encoders with software.
  • FossFLOW - create cracking looking network diagrams

// what's coming ------------------

Hosted SlingShot, hardware bundles, and proper support for teams who'd rather not run it themselves.

Specialist hardware, allowing you to get the job done regardless of the requirements.
